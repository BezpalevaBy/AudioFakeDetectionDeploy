# AudioDeepFake Detection Приложение

Инструкция по запуску:
- Установить Docker Desktop (Windows) / установить пакет docker compose (Ubuntu)
- git clone [Ссылка на репозиторий]
- В командой строке проекта выполнить docker compose up -d
-  Дождаться запуска контейнера, открыть в браузерной строке http://localhost:8000

Запуск контейнера долгий, работа выполнена в полном объёме с одним исключением - детальные признаки фейков. Там требуется ручная работа человека с подходящими инструментами, по типу adobe audition и прочее.
